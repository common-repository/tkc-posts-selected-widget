This plugin allows you show list of selected posts via widget.

== Description ==

Displaying selected posts via widget.

== Installation ==

1. Upload the entire 'tkc-posts-selected' directory to the '/wp-content/plugins/' directory.
2. Activate TKC Posts Selected through the 'Plugins' menu in WordPress.
